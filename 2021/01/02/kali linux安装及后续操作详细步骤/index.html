    进入系统后的后续操作

    一、检查软件更新及安装增强工具

    1. 打开终端，输入 sudo su- 切换成管理员身份
    2. 输入 apt update 查询官网下的软件是否为最新版
    3. 输入 apt -y full-upgrade 进行更新软件
    4. 更新完成后，输入 apt -y install open-vm-tools-desktop fuse 下载增强工具（kali 官方推荐），我也不知道具体的作用
    5. 运行 reboot 重启

      二、安装拼音输入法

    6. 使用 apt-get install fcitx-googlepinyin 安装谷歌拼音
    7. 使用 apt-get install fcitx-table-wbpy 安装拼音、双拼、五笔拼音输入法
    8. 1和2自己看情况可以选择安装，然后使用 reboot 重新启动
    9. 使用Ctrl+空格，可以调出输入法。
    10. 使用Ctrl+Shift，可以切换输入法。

      三、屏幕时不时闪烁，应该是显卡的原因

      运行 sudo add-apt-repository ppa:bumblebee/stable
      再运行 sudo apt-get update
      然后运行 sudo apt-get install bumblebee bumblebee-nvidia
      然后重启 reboot
